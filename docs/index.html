Instalação rápida

Comandos recomendados para instalação limpa. Use exatamente assim no Termux.

1) Atualizar e instalar git 
pkg update -y && pkg upgrade -y
pkg install -y git
2) Clonar o projeto 
git clone https://github.com/Anderson007s/HyperHDR-Termux.git
cd HyperHDR-Termux
3) Rodar o instalador
recomendado
. 
bash install-termux.sh
Dica: se algum pacote falhar por espelho, rode termux-change-repo e selecione um mirror estável.

Build e execução

Após instalar dependências, você pode compilar e rodar usando os scripts.

Compilar (release) 
./build.sh
Rodar (foreground) 
./run-hyperhdr.sh

O que este projeto é?

Um conjunto de scripts e ajustes para facilitar o uso do HyperHDR no Termux. Se você quer Ambilight via rede (ex.: WLED) e players compatíveis com captura no Android, este repo ajuda a deixar o ambiente pronto.

Observação: apps com DRM (Netflix/Prime/Disney etc) normalmente bloqueiam captura — isso não é limitação do projeto, e sim do Android/DRM.
